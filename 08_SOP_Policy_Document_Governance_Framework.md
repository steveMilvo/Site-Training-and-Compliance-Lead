# 08 - SOP, Policy & Document Governance Framework
## Assessment Question
### "How would you improve compliance, consistency and operational performance?"

---

# Introduction

One of the largest hidden risks in any manufacturing organisation is not a lack of procedures.

It is multiple versions of procedures.

Different departments often develop excellent documentation independently.

Over time however this can result in:

- Duplicate SOPs
- Outdated procedures
- Different versions of the same document
- Employees unsure which version is current
- Audit findings
- Training inconsistencies
- Operational variation

My objective would be to create a robust document governance framework that provides confidence that every employee is always working from current approved information.

---

# My Philosophy

Good documentation should be:

Easy to find.

Easy to understand.

Easy to maintain.

Easy to audit.

Easy to train.

Easy to review.

If employees cannot quickly find the correct document, the governance system has failed.
